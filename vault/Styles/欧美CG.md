# 欧美CG

> 3D渲染质感、写实光影、电影感。适合科幻、豪门、职场、动作题材。

## 风格词（全局标签）

```
Western CG animation, Pixar-quality rendering, realistic shading, vibrant colors, 3D look, high quality, masterpiece
```

## 核心视觉特征

- **渲染**：3D建模感，有体积感和材质反射
- **光影**：电影级打光，阴影细腻
- **色彩**：高对比，色彩饱和但不失真实感
- **人物**：比例略夸张（大眼、小鼻），但有真实皮肤质感
- **背景**：精细建模，有景深效果

## 两种子风格

### Pixar 风（圆润、温暖）
```
Pixar animation style, 3D rendered, smooth surfaces, warm lighting, subsurface scattering skin, vibrant colors
```

### 电影CG风（写实、硬朗）
```
cinematic CG rendering, realistic 3D, dramatic lighting, photorealistic textures, 8K quality, film production quality
```

## 人物提示词

```
White background. Three views: front | side | back. Same character, same height.
[角色描述]
Western CG animation style, Pixar-quality 3D rendering, realistic shading, detailed facial features.
high quality, character model sheet, turnaround
```

注意：欧美CG风的三视图更接近3D模型转台图（turnaround），可加 `turnaround view` 增强。

## 场景提示词

```
Wide establishing shot. [场景名]. [室内/室外]. [时间/光线].
[场景描述]
Western CG animation, cinematic 3D rendering, realistic lighting, detailed environment, no characters.
high quality, masterpiece
```

## 豪门/职场场景专用词

```
luxury interior, marble floors, floor-to-ceiling windows, city skyline, modern architecture
glass and steel office building, executive penthouse, grand lobby, helipad
```

## 常见问题

| 问题 | 修复 |
|------|------|
| 太像游戏截图 | 去掉 `game` 相关词，加 `film production, movie quality` |
| 皮肤太塑料感 | 加 `subsurface scattering, realistic skin texture` |
| 光线太平 | 加 `dramatic lighting, strong shadows, rim light` |
