# 吉卜力

> 手绘感强、暖色调、圆润可爱。适合奇幻、家庭、冒险题材。

## 风格词（全局标签）

```
Studio Ghibli style, hand-drawn animation, warm earthy tones, whimsical atmosphere, soft colors, high quality, masterpiece
```

## 核心视觉特征

- **线条**：手绘感强，有轻微抖动，不是完美直线
- **色彩**：暖黄、草绿、土棕为主，蓝天纯净
- **质感**：水彩纸张感，有轻微颗粒
- **角色**：圆眼睛、圆脸，表情夸张但真诚
- **背景**：植物/自然元素丰富，建筑有欧式童话感

## 人物提示词

```

```

## 常见问题

| 问题 | 修复 |
|------|------|
| 太像宫崎骏原作角色 | 加 `original character, not from any existing Ghibli film` |
| 背景过于写实 | 加 `painterly style, illustrated, not photorealistic` |
| 颜色太鲜艳 | 加 `muted warm tones, soft palette, desaturated` |
