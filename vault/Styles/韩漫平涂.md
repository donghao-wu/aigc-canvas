# 韩漫平涂

> 当前最常用画风。适合都市、甜宠、逆袭题材。

## 风格词（全局标签）

```
Korean webtoon style, flat color illustration, clean bold linework, manhwa aesthetic, vibrant palette, high quality, masterpiece
```

## 人物提示词最佳实践

### 三视图

```
White background. Clean linework. Three views: front | side | back. Same character, same clothes, same height across all three. Korean webtoon style, flat color illustration, clean bold linework, manhwa aesthetic. [角色描述]. high quality
```

## 常见问题与修复

| 问题 | 修复方法 |
|------|---------|
| 线条粗细不一致 | 加 `uniform linework, consistent stroke weight` |
| 颜色过暗 | 加 `bright colors, high saturation, vivid` |
| 三视图角度不对 | 加 `strictly front view / strictly side profile / strictly back view` |
| 出现真人感 | 加 `2D illustration, not photorealistic, anime style` |
