# 新海诚

> 光感强、背景精细、青春感。适合青春、纯爱、日常题材。

## 风格词（全局标签）

```
Makoto Shinkai anime style, soft watercolor sky, detailed backgrounds, natural lighting, cel-shaded, high quality, masterpiece
```

## 核心视觉特征

- **光晕**：光线从窗口/缝隙渗入，形成强烈光晕（lens flare）
- **背景密度**：背景细节极高，远比角色复杂
- **色彩**：蓝紫色系为主，金色光点点缀
- **角色简化**：角色相对背景是"简化版"，cel-shaded 平涂感

## 人物提示词注意

新海诚风格中人物不是视觉重心。三视图时背景需留白：
```
White background. [人物描述]. Makoto Shinkai character design, cel-shaded anime style, soft colors, clean linework. Three views: front | side | back. Same character, same height.
```

## 场景提示词最佳实践

场景是新海诚风格的强项：
```
Wide shot. [场景名]. Makoto Shinkai anime style, photorealistic background, soft watercolor sky, golden hour light, lens flare, detailed environment, no characters. high quality, masterpiece
```

## 关键词增强

```
ray of light, bokeh, dramatic lighting, atmospheric perspective, detailed cityscape
```
