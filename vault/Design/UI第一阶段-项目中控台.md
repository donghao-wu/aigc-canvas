# UI 第一阶段：项目中控台

日期：2026-05-04

## 目标

把项目首页从“项目列表”改为“生产中控台”，先建立整个平台的新视觉方向，再逐步迁移剧本工作台、画布和资产库。

## 已实现

- 左侧 Studio 导航：生产总览、项目队列、剧本流水线、视觉资产、视频交付
- 顶部 Production Command 区：标题、主题切换、退出、新建项目
- 统计卡：项目、资产、图片、Token、费用
- 项目队列：搜索、流水线进度、成员、节点数、更新时间、重命名、删除
- 右侧生产流：最近项目按更新时间展示当前阶段
- 全局资产速览：类型筛选、资产缩略图、资产预览、复制 Prompt

## 设计原则

- 更像专业制作工具，不像营销页
- 信息密度高，但每个模块仍保持清晰边界
- 深色为主，浅色保留可用
- 卡片圆角控制在 8-12px
- 使用 lucide 图标替代文字装饰和 emoji

## 下一步

- 项目内统一顶部 App Shell，替换剧本/画布/资产库各自的旧 header
- 剧本工作台改为左步骤栏 + 中心文档区 + 右侧生成控制/版本面板
- 画布增加左工具栏和右 Inspector
- 资产库增加密集筛选、批量生成和资产使用时间线
