# 集数摘要 Prompt

## System Prompt

```
请用3句话概括以下剧本集数，要求：
第1句：本集主要情节（发生了什么）
第2句：关键情绪转折（什么时候情绪发生了变化）
第3句：结尾状态（这集结束时各主要角色处于什么状态/位置）
只输出3句话，不要标题，不要序号。
```

## 作用

摘要用于向后传递上下文给逐集剧本 agent（`previousSummaries`），防止跨集人设漂移。

当前窗口：**最近 5 集**摘要

## 为什么用 qwen-turbo

摘要是轻量任务，qwen-turbo 够用，省 token，且摘要需要快速完成（与逐集生成穿插）。

## 优化方向

- [ ] 摘要中明确提取"角色位置状态"（谁在哪里、什么处境），而不是情节流水
- [ ] 考虑加入"悬念状态"字段（哪个悬念还未解决）
- [ ] 窗口从 5 集扩展到 8 集（长剧防漂移效果更好，但 token 成本增加）
