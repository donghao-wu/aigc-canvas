# 短剧爽点结构

> 研究短视频平台短剧的留存规律，指导故事圣经和集数大纲的设计。

## 核心留存公式

```
留存 = 钩子强度 × 爽感密度 × 悬念牵引
```

三者缺一不可：
- **钩子强度**：前5秒能不能让人停下来
- **爽感密度**：平均每N集有一个让人拍桌子的爽点
- **悬念牵引**：每集结尾让人忍不住点下一集

## 钩子类型

| 类型 | 描述 | 适用题材 |
|------|------|---------|
| 打脸钩 | 主角当众被羞辱/误解，观众急于看反转 | 逆袭、都市 |
| 身份钩 | 真实身份被隐藏，观众期待揭秘 | 豪门、悬疑 |
| 情感钩 | 两人关系悬而未决，观众期待表白/分手 | 甜宠、虐恋 |
| 危机钩 | 主角陷入绝境，观众期待脱困 | 悬疑、热血 |
| 反转钩 | 上集结尾埋下的局，这集开始兑现 | 所有题材 |

## 爽点节奏建议

**60集剧的爽感节点参考：**

| 集数区间 | 爽感类型 | 强度 |
|---------|---------|------|
| 1-5集 | 小爽（主角小反击） | ★★ |
| 10-15集 | 中爽（中反派第一次吃亏） | ★★★ |
| 20-25集 | 大爽（主角完成阶段性逆袭） | ★★★★ |
| 30集 | 最大反转（不是爽而是震惊） | ★★★★★ |
| 40-45集 | 大爽（中反派彻底落败） | ★★★★ |
| 55-58集 | 每集强爽（终局对决） | ★★★★★ |
| 大结局 | 圆满落定 | 情感满足 |

## 悬念类型

| 类型 | 维持周期 |
|------|---------|
| 集内悬念（本集结尾）| 1集 |
| 短线悬念 | 3-5集 |
| 中线悬念 | 10-15集 |
| 长线悬念 | 30集以上 |

每条长线悬念必须在 40-49 集收束（见[[Agents/01-故事圣经]]节拍铁律）。

## 对白爽感技巧

**打脸台词结构：**
```
反派：[嘲讽/轻视，具体到某件事]
主角：[反将一军，直击对方弱点]
△ [在场人反应]
```

**揭秘台词结构：**
```
△ [道具/文件/现场出现]
角色A：（惊讶）这是……
角色B：（冷/坚定）对。[一句话说清楚真相]
△ [对方瞬间的身体反应]
```

## 题材爽感特征

| 题材 | 核心爽感 | 观众情绪 |
|------|---------|---------|
| 逆袭 | 打脸 + 身份揭秘 | 爽 + 解气 |
| 甜宠 | 互动甜蜜 + 误会解除 | 甜 + 心跳 |
| 悬疑 | 线索揭秘 + 反转 | 震惊 + 成就感 |
| 豪门 | 身份逆转 + 权力游戏 | 爽 + 窒息感 |
| 古装 | 打脸 + 上位 | 爽 + 痛快 |
