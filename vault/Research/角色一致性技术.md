# 角色一致性技术

> 漫剧制作最核心的挑战：让同一个角色在几十甚至几百张图里保持一致。

## 一致性问题的根源

文生图模型每次生成都是独立的随机过程。
即使 prompt 完全一样，结果也会有差异。

一致性 = **锚定描述** + **参考图** + **固定句**，三者缺一不可。

---

## 方法一：Character DNA（文字锚定）

给每个角色写一段固定的英文描述（DNA），每次生图都带上。

### DNA 格式

```
[性别], [年龄描述], [体型], [发型+发色], [眼睛], [皮肤], [标志性服装]
```

### 示例

```
林若曦-DNA:
Female, mid-20s, slim and tall, long straight black hair, sharp almond eyes, fair skin, usually wearing white office blazer with pencil skirt

秦霍-DNA:
Male, early 30s, athletic build, short dark hair slightly disheveled, deep-set eyes, strong jawline, usually wearing dark business suit
```

### 使用方式

把 DNA 拼接到任何涉及该角色的提示词开头：
```
[角色DNA], [当前场景描述], [动作/表情], [画风标签]
```

### 实现计划

见 [[Agents/03-资产登记]] 的优化方向——在资产登记 prompt 中加入 `DNA:` 字段。

---

## 方法二：参考图（图像锚定）

NanoBanana / GPT Image 2 都支持参考图输入，这是最强的一致性保证。

### 工作流

```
1. 生成"母图"（三视图或正面标准图）
2. 后续所有图以母图为参考图输入
3. 固定句：
   "Same character as reference image.
    Change only: [你想改变的内容].
    Keep all character features unchanged."
```

### 母图优先级

```
三视图参考表 > 正面标准图 > 半身像
```

三视图包含最完整的角色信息（正面/侧面/背面），应作为第一张母图。

---

## 方法三：固定句库

一些经过验证的固定句，粘贴进 prompt 即可增强一致性。

### 角色一致性固定句

```
// 基础锁定
Same character as reference image. Same art style, same character design.

// 只改表情
Same character. Same clothes. Same pose. Change only: facial expression to [情绪].

// 只改角度
Same character as reference. Change only: camera angle to [角度]. Keep character design identical.

// 只改服装
Same character. Same face, same hairstyle. Change only: outfit to [新服装描述].

// 角色置入场景
Place the character from [image 1] into the scene from [image 2]. Keep character design unchanged.
```

### 场景一致性固定句

```
// 基础锁定
Maintain the same environment, lighting, color palette, and art style as the reference image.

// 只改角度
Maintain the same scene. Change only: camera angle to [角度].

// 加入角色
Same scene as reference. Add a character: [角色描述]. Keep background unchanged.
```

## 方法四：LoRA / 微调（高阶）

如果角色需要在 100+ 张图里保持高度一致，可以考虑：
- 训练角色 LoRA（需要 10-20 张种子图）
- 用 ComfyUI + IP-Adapter 实现精确面部控制

**当前阶段不需要**，参考图方法对漫剧制作已经足够。

---

## 一致性检查清单

每次生成新图时对照：

- [ ] 发型颜色是否一致？
- [ ] 脸型是否一致？（韩漫风容易漂移）
- [ ] 眼睛形状是否一致？
- [ ] 服装细节是否对得上？
- [ ] 画风标签是否与其他图一致？
- [ ] 体型比例是否一致？

| 画风 | 一致性难度 | 主要漂移部位 |
|------|-----------|------------|
| 韩漫平涂 | ★★★ | 脸型、眼睛形状 |
| 新海诚 | ★★ | 整体风格稳定，细节有差异 |
| 吉卜力 | ★★ | 体型比例 |
| 国风漫画 | ★★★★ | 服装细节、发饰 |
| 欧美CG | ★★★ | 皮肤质感、光影方向 |
| 写实 | ★★★★★ | 面部特征 |
