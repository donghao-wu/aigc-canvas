# 人物特写模板

> 用于生成角色半身像/头像，展示表情和面部特征。
> 通常以三视图中的正面图为参考图输入，确保一致性。

## 通用模板

```
Close-up portrait, upper body.
[角色描述]
[情绪/表情关键词]
[画风标签]
soft lighting, detailed face, high quality
```

## 情绪关键词速查

| 情绪（中文）| 英文关键词 |
|------------|-----------|
| 冷漠/高冷 | cold expression, emotionless gaze, distant look |
| 微笑/温柔 | gentle smile, warm expression, soft eyes |
| 愤怒 | angry expression, furrowed brows, intense gaze |
| 惊讶 | surprised expression, wide eyes, open mouth |
| 委屈/快哭 | teary eyes, sad expression, trembling lips |
| 坚定 | determined expression, strong gaze, set jaw |
| 讽刺/冷笑 | smirk, sardonic smile, raised eyebrow |
| 绝望 | desperate expression, hollow eyes, pale face |
| 喜悦 | joyful expression, bright eyes, wide smile |

## 使用参考图时的固定句

以三视图的正面为参考图：
```
Same character as reference image. Close-up portrait, upper body only.
Change only: facial expression to [情绪关键词].
Same art style, same character design, same clothing.
```

## 特写的构图变体

| 构图 | 说明 | 英文 |
|------|------|------|
| 头像 | 只到肩膀 | `headshot, face and neck only` |
| 半身 | 到腰部 | `upper body portrait, waist up` |
| 四分之三 | 稍微斜侧 | `three-quarter view portrait` |
| 正面 | 完全正对 | `front-facing portrait, direct gaze` |
| 侧脸 | 侧45度 | `three-quarter profile, slight turn` |

## 最佳实践

1. **先有三视图，再出特写**——三视图是设定，特写是演绎
2. **情绪词要具体**——`sad` 不如 `teary eyes, trembling lips`
3. **背景虚化**——特写背景用 `soft bokeh background` 或 `simple gradient background`，避免复杂背景抢镜
