# 人物三视图模板

> 用于生成角色参考表（Reference Sheet），正面/侧面/背面三视图。
> 这是角色一致性的基础，后续所有该角色的图都应以此为参考图。

## 通用模板

```
White background. Clean linework. Minimalist. 
Three views: left section = full body front view | center = full body side profile | right = full body back view. 
Same character, same clothes, same height across all three views.
[角色描述]
[画风标签]
high quality, character design sheet, reference sheet
```

## 角色描述填写规范

```
[性别] character, [年龄段] years old, [体型描述], [发型+发色], [服装描述]
```

### 示例
```
Female character, mid-20s, slim build, long straight black hair, wearing white office blazer and pencil skirt
```

## 常见失败原因

| 失败情况 | 原因 | 修复 |
|---------|------|------|
| 三个视角角度错 | 模型理解"side"不一定是90° | 加 `strictly side profile (90 degrees)` |
| 三个视角角色不一致 | 模型没锁定"same character" | 前置强调 `Identical character in all three views` |
| 背景有颜色 | 没强调白底 | 加 `pure white background, no shadow` |
| 有多余角色 | 提示词描述模糊 | 加 `single character only` |

## NanoBanana 特殊技巧

NanoBanana 支持参考图输入。三视图生成流程：
1. 先用文字生成正面图
2. 以正面图为参考，生成侧面（加 prompt：`Same character as reference. Side profile view only.`）
3. 以正面图为参考，生成背面
