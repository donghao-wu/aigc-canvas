# 场景多角度模板

> 每个场景生成4个角度，建立"场景一致性"。
> 生成顺序：全景 → 中景 → 细节 → 俯视。
> 建立镜头（全景）是基准图，其他角度应以它为参考图输入。

## 四角度模板

### ① 全景（建立镜头）·基准图

```
Wide establishing shot. [场景名]. [室内/室外] [时间段].
[场景描述，含主要陈设和氛围]
No characters, empty scene.
[画风标签]
cinematic composition, high quality, masterpiece
```

### ② 中景（场景内部）

```
Medium shot, interior view. [场景名]. [时间段].
[重点陈设描述，如桌椅/窗户/主要道具]
[画风标签]
Maintain the same environment, lighting, color palette, and art style as the reference image. Change only: camera angle to medium shot.
high quality
```

### ③ 细节特写

```
Detail close-up shot. [具体物件或纹理描述，如：wooden desk with scattered papers / marble floor tile].
[画风标签]
Maintain the same art style and color palette as the reference image.
high quality, detailed texture
```

### ④ 俯视

```
Aerial bird's-eye view. [场景名]. Looking straight down.
[场景平面布局描述]
[画风标签]
Maintain the same environment and art style as the reference image. Change only: camera angle to aerial view.
high quality
```

## 场景一致性关键句

在 ②③④ 的提示词中加入这句话，然后上传全景图作为参考：
```
Maintain the same environment, lighting, color palette, and art style as the reference image.
```

## 室内 vs 室外注意点

**室内场景**
- 光源要明确：`natural light from window` / `warm artificial lighting`
- 禁止写"光线柔和"这种模糊描述，要写光源位置

**室外场景**
- 时间段影响极大：日 = `daylight, blue sky` / 夜 = `night, city lights, dark sky`
- 加入天气状态：`clear sky` / `overcast` / `light rain`

## 示例：现代办公室·韩漫风

```
全景：Wide establishing shot. Modern corporate office. Indoor, daytime.
Glass partition walls, rows of desks with monitors, city skyline visible through floor-to-ceiling windows.
No characters, empty scene.
Korean webtoon style, flat color illustration, clean bold linework, manhwa aesthetic, vibrant palette.
cinematic composition, high quality

中景：Medium shot, interior view. Corporate office. Daytime.
Executive desk, leather chair, computer monitor, city view window behind.
Korean webtoon style, flat color illustration, clean bold linework.
Maintain the same environment, lighting, color palette, and art style as the reference image.
high quality
```
